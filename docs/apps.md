- [Jetbrains Toolbox](https://www.jetbrains.com/help/toolbox-app/installation.html#tba_installation)
    ```
    mkdir ~/Applications
    cd ~/Applications
    tar -xvf jetbrains-toolbox-<version>.tar.gz -C <installation-directory>
    ```
